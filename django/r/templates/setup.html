Setup the R App

This app depends R, which is an external library for statistical computation. Please download and install the library for your OS:

Download For Mac Download For Windows

After installing, please check the installation path. For Windows, it should be something like C:\Program Files\R\R-3.0.3\bin\R.exe. For Mac OS X, it's probably /Library/Frameworks/R.framework/Versions/Current/Resources/bin/R.
